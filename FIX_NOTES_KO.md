# 수정 반영 내역

## 1. 임원 요약 오류 원인 표시

기존에는 임원 요약 생성 실패 시 프론트에서 `결론을 생성하지 못했습니다.`로만 표시되어 실제 원인이 가려졌습니다.

수정 후에는 Gemini 빈 응답, JSON 파싱 실패, API 호출 실패, verdict 누락을 구분해 `/api/sessions/{id}/executive_summary`가 실제 원인을 반환합니다.

## 2. Quick Win 요약

임원 보고용 요약에서 Quick Win 액션이 있으면 결론 아래에 `⭐ 지금 바로 할 일` 요약 박스가 표시됩니다.

## 3. 세션 전체 분석 신뢰도

발언별 신뢰도 숫자는 오해 소지가 있어 제거하고, 세션 전체 기준의 `분석 신뢰도 N%`를 추가했습니다.

신뢰도는 정답 확률이 아니라 아래 요소를 종합한 분석 신뢰 점수입니다.

- 데이터 수집 품질
- 근거 충실도
- AI 응답 안정성
- 추론 비중 리스크
- 외부 검증 가능성

우측 패널과 임원 요약 모달에 함께 표시됩니다.

## 4. 업로드 구조

폴더 없이 GitHub 웹 업로드가 가능하도록 모든 분할 파일은 root에 위치합니다.

- `js_00_state.js` 형태
- `css_00_root_base.css` 형태
- `/assets/{asset_name}` 경로로 서빙


## 4. 우측 패널 순서 조정

토큰/LLM 문제와 무관하게 우측 패널에서 검증 메모가 요약보다 먼저 보이던 UI 순서를 수정했습니다. 이제 `요약 → Sharp Insights → 검증 메모 → 축별 점수/디테일` 순서로 표시됩니다.


## YouTube 분석 범위 문구 명확화

- YouTube URL 입력 시 현재 앱은 가능한 경우 자막/자동자막과 페이지 메타데이터를 분석합니다.
- 영상 화면/장면 자체를 Gemini Vision으로 분석하는 기능은 현재 포함하지 않습니다.
- 이번 수정은 기능 로직 변경 없이 UI/문서 문구만 명확히 한 것입니다.


## 이번 최종 추가 반영

- `BrandStrategist` 마케팅 페르소나 추가
- AI 1차 진단 결과에 `브랜드 적합도` 점수/근거/개선 액션 추가
- 우측 패널, AI 1차 진단, 임원진 보고용 요약에 브랜드 적합도 카드 표시
- 브랜드 적합도는 Samsung Galaxy / Apple 공식 브랜드 아이덴티티와 브랜드 페르소나 기준이며, 실제 시장 반응·검색량·구매 의향은 포함하지 않음
- 기존 DB에 기본 페르소나가 이미 있어도 새 기본 페르소나가 누락되어 있으면 자동 보강


## 브랜드 페르소나 가중치 조정

- 브랜드 적합도 가중치: 공식 브랜드 아이덴티티 40점 / 브랜드 페르소나 45점 / 근거 명확성 15점
- 화면의 `브랜드 페르소나 판단 기준 보기`는 기본 접힘 상태로 표시됨


## 이번 추가 수정
- 새 토론 기본 선택 페르소나를 BrandStrategist에서 AICommerceHacker로 변경했습니다.
- 브랜드 메시지 적합도는 데이터/콘텐츠/AI Commerce/UX와 같은 5개 축 중 하나로 표시됩니다.

## 2026-05-12 추가 수정

- 새 토론 모달에서 페르소나/라운드/모드 변경 시 입력해둔 토론 주제, URL, 카피 문구, A/B 라벨·URL이 유지되도록 수정했습니다.
- 토론 주제/URL preset은 사용자 컨펌 전 단계라 반영하지 않았습니다.
- URL/파일 없이 카피 문구만 입력해도 세션 생성 및 5축 진단이 가능하도록 `copy_text` 입력을 추가했습니다.
- 카피 문구만 분석할 경우 페이지 구조, 이미지, UX 데이터는 제한적으로 평가된다는 안내 문구를 추가했습니다.
